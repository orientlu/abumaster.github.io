左值和右值 分发

左值：是即能放在等号的左边又能放在等号的右边，是在内存中有实体的存在，占有内存空间；
右值：是只能放在等号的右边，不能被赋值，只是一个临时变量，存放在寄存器中，没有分配存储空间。

左值引用 使用 Type &左值引用名 = 左值表达式; 是左值的一个别名，本质是一个指针；
右值引用 用法 Type &&右值引用名 = 右值表达式; 由于右值在内存中没有具体的内存空间，所以不能复制给左值引用，使用右值引用，如在函数的声明过程中
